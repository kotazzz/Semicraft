# Armor Stand (Ожившая стойка)

Мирная аномальная сущность на сервере Semicraft.

| Поле | Значение |
| --- | --- |
| Тип | Аномальный объект |
| Угроза | Низкая |
| Поведение | Мирное, помогает |

## Поведение

- Перемещается, когда игроки не смотрят
- Оставляет **деревянные таблички** с подсказками
- Помогает с навигацией и скрытыми проходами
