# Introduction

Semicraft is an ARG about a Minecraft server "infected" with **bugs**.

## Timeline

| Era | Event |
| --- | --- |
| **2015** | Server opened June 2, 2015 in beta; HanoiHaze commissioned an Alpha-style resource pack |
| **2015** | Shutdown due to bugs |
| **2020** | Reopened and shut down again |
| **2026** | Server active again; bugs continue |

## Key Elements

- **[[en/lore/Terminal|Terminal]]** on [semicraft.su](http://semicraft.su) — passwords are usually player nicknames
- **[[en/lore/C.Y.|C.Y.]]** — AI maintaining the terminal
- **[[en/lore/C.I.|C.I.]]** — former terminal AI, disappeared for unknown reasons
- **[[en/anomalies/Virus|The Virus]]** — player illness; physical manifestation — [[en/anomalies/Star|Star]]
- **[[en/lore/Rituals|Rituals]]** — structures and sacrifices with unpredictable outcomes
- **[[en/lore/The Void|The Void]]** — space outside the server, Star's home

Source: *Semicraft Glossary* (MWD Archive, 2026).
