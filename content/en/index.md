# Semicraft

**Semicraft** is an ARG about a Minecraft server "infected" with bugs. The project started in **2015**; the server was shut down due to anomalies in **2015** and **2020**, then reopened. It is now **2026** — bugs and oddities continue.

A public research server with mystery elements, visually mimicking early Alpha versions of Minecraft.
