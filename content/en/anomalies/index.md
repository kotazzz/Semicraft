# Semicraft Server Anomalies

The world of Semicraft is subject to system errors, generation glitches, and unexplained phenomena — **anomalies**. Many mimic bugs from early Alpha versions; some are tied to the project's lore.

## Diseases and Phenomena

### [[en/anomalies/Virus|The Virus]]

A player illness with multiple stages. Physical manifestation — **[[en/anomalies/Star|Star]]**. Protection: pumpkins (![sign](images/community/sign_vietnamese_pumpkin.png)).

### [[en/anomalies/Mind Control|Mind Control]]

Loss of character control under external forces.

## Entities

### [[en/anomalies/Star|Star]]

Phantom / particle cloud. **Destroyed by C.Y.** Bribed players, pushed them toward [[en/lore/Rituals|rituals]].

### [[en/anomalies/Armorstand|Armor Stand]]

Peaceful entity; helps via signs (![appearance](images/community/sentient_armorstand_appearance.png)).

## Technical Anomalies

### [[en/anomalies/Terrain Glitches|Terrain generation glitches]]

Chunk slices, floating islands — led to server shutdowns in 2015 and 2020.

### [[en/anomalies/System Glitches|System glitches]]

Strange client and server behavior.

### AI Activity

- **[[en/lore/C.Y.|C.Y.]]** — terminal, system logs
- **[[en/lore/C.I.|C.I.]]** — former terminal operator
