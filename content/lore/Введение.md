# Введение

Semicraft — ARG о Minecraft-сервере, «заражённом» **багами**.

## Хронология

| Эпоха | Событие |
| --- | --- |
| **2015** | Сервер открыт 2 июня 2015 в beta; HanoiHaze поручил создать resource pack в стиле alpha |
| **2015** | Закрытие из‑за багов |
| **2020** | Повторное открытие и повторное закрытие |
| **2026** | Сервер снова активен; баги продолжаются |

## Ключевые элементы

- **[[lore/Терминал|Терминал]]** на [semicraft.su](http://semicraft.su) — пароли обычно = никнеймы игроков
- **[[lore/C.Y.|C.Y.]]** — ИИ, поддерживающий терминал
- **[[lore/C.I.|C.I.]]** — прежний ИИ терминала, исчез по неизвестным причинам
- **[[anomalies/Вирус|Вирус]]** — болезнь игроков; физическое воплощение — [[anomalies/Star|Star]]
- **[[lore/Ритуалы|Ритуалы]]** — постройки и жертвоприношения с непредсказуемым исходом
