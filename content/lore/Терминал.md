# Терминал

Скрытый раздел на [semicraft.su](http://semicraft.su): клик по **логотипу** на главной странице.

Экран ввода пароля возвращает записи о игроках, событиях и лоре. Поддерживается **[[lore/C.Y.|C.Y.]]** (версии v0.1 / v0.2, зелёная тема). Записи **[[lore/C.I.|C.I.]]** используют красную тему «TERMINAL 42».

Полная таблица паролей: [[site/Сайт Semicraft#Известные пароли|Сайт Semicraft → Известные пароли]].

## Примеры записей

| Пароль | Лог |
| --- | --- |
| `Semicraft` | Сервер открыт 2 июня 2015; Hanoi поручил resource pack в стиле alpha |
| `Bobbies1` | YouTube-канал + Google Drive с информацией о сервере |
| `Hanoi` | Создатель из Вьетнама, любит кошек; в UPDATE: *He hates C.I.* |
| `C.Y.` | *It's ME, I was created by Hanoi* |
| `Stars` | *He's a damn virus.* |
| `SlimeyPretty` | Хороший человек, но проводит ритуалы — *I DON'T LIKE THIS* |
| `kot_az` | *one of the smartest in the orden, he poses a real danger to ME* |
